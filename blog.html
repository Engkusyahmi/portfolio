POST #1 • IOT ENGINEERING

Building AquaMonitor: Integrating ESP32 with Flutter

Published on: 2025

Developing the AquaMonitor system required a tight integration between hardware and mobile app software. Using the Agile Scrum methodology, I continuously validated secure data transmission between the ESP32 microcontroller and the Flutter mobile application. Designing complex database routing using Crow's Foot Notation ERDs and DFDs ensured that live water flow data from the YF-S201 sensor streams smoothly into our hosted MySQL database via custom PHP native APIs.

POST #2 • WEB APPLICATION

Saku Legacy: Handling Web-Based Payments via ToyyibPay

Published on: 2025

For the Saku Legacy Photo Booth application, the biggest technical challenge was creating a frictionless, responsive mobile browser user experience without requiring an application store install. By implementing a custom QR-code access flow, users can launch guest mode instantly. This post details the integration of the ToyyibPay online payment gateway to authorize secure transactions before digital image assets are customized and submitted.
