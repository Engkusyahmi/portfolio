## 🛠️ 2. Core Featured Projects

### 🌟 Project 1: AquaMonitor – Smart Water Monitor System
*   **Role:** Final Year Project (Individual Core Developer)
*   **Tech Stack:** Flutter, Dart, PHP (Native RESTful API), Hosted MySQL Database, ESP32 DevKit V1, YF-S201 Water Flow Sensor.
*   **Key Engineering Features:**
    *   **IoT System Architecture:** Real-time data streaming from YF-S201 hardware flow sensors into cloud storage via an ESP32 microcontroller.
    *   **Mobile Dashboard:** Built a highly interactive cross-platform Flutter application for structural live-data visualization.
    *   **AI Integration:** Integrated an automated AI Chatbot inside the app to provide predictive usage insights and user query resolutions.
    *   **Methodology & Analysis:** Designed detailed software logics using Crow's Foot Notation ERDs and Data Flow Diagrams (DFD Level 0 & Level 1) optimized under an Agile (Scrum) software deployment process.

### 📸 Project 2: Saku Legacy – Photo Booth Magnet App
*   **Role:** Individual Project Developer
*   **Tech Stack:** Flutter Web, Dart, ToyyibPay Gateway, Android Studio.
*   **Key Engineering Features:**
    *   **Frictionless UX:** QR-code-based application routing allowing guest users to capture and render photo frames instantly on mobile browsers without app installations.
    *   **FinTech Core:** Clean integration of ToyyibPay online payment gateway workflows to process secure web transactions before final asset rendering.

### 🚗 Project 3: Drive N Rent – Car Rental System
*   **Role:** Lead System Architecture & Backend Developer
*   **Tech Stack:** Native PHP, MySQL Database, Apache Server, Bootstrap 5, VS Code.
*   **Key Engineering Features:**
    *   Formulated technical data schemas and solved system interface bottlenecks for a practical management dashboard.
    *   Executed code debugging and modular code component optimization to secure server execution speeds.
